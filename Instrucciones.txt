INSTRUCCIONES PRÁCTICA 1

Desarrollo de la práctica
Esta primera práctica consiste en:
- Desde la cuenta de Github hacer un "fork" del repositorio: https://github.com/gitt-3-pat/p1. 
- Probar los siguientes comandos sobre el anterior repositorio:
    - git clone
    - git status
    - git add
    - git commit
    - git push
    - git checkout
- Dejar en el repositorio un documento (git.txt o git.pdf) explicando los comandos anteriores: qué significan, para qué los has usado, logs que han dejado en la consola tras su ejecución.
